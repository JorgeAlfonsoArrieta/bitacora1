IDEAS

-Función: Se utilizan para mostrar contenido visual en páginas web, como fotos, gráficos y logotipos.

- Elemento < img>: Se inserta con el elemento < img>, que no necesita etiqueta de cierre.

- Atributo src: Especifica la ruta o URL de la imagen.

-Atributo title: Proporciona información adicional que aparece como un tooltip cuando se pasa el cursor sobre el enlace.

-Atributo alt: Proporciona un texto alternativo que describe la imagen, esencial para la accesibilidad.

APUNTES

Las imágenes en HTML se utilizan para mostrar contenido visual en las páginas web, como fotos, gráficos y logotipos. Se inserta mediante el elemento `< img >`, que no requiere etiqueta de cierre. El atributo `src` especifica la ruta o URL de la imagen, mientras que el atributo `alt` proporciona un texto alternativo que describe la imagen, crucial para la accesibilidad y para situaciones donde la imagen no se carga.
