APUNTES

HTML (HyperText Markup Language) es el componente más básico de la web utilizado para crear y diseñar páginas web. Define con etiquetas el significado y la estructura del contenido web de forma organizada.

Utiliza los simbolos <> para etiquetar texto,imagenes y otro contenido mostrado en un navegador web. Utiliza elementos especiales que significan algo para el navegador , la gran mayoria d estos deben tener principio y fin.

RESUMEN

HTML (HyperText Markup Language) es el lenguaje básico que se utiliza para crear páginas web. Es la estructura o los cimientos de un edificio: no se ve bonito, pero es lo que hace que todo se sotenga.

Es como un conjunto de instrucciones que se le da a un navegador para que muestre una página web. Usas etiquetas (como < p > para párrafos o < img > para imágenes) que le indican al navegador qué debe hacer con cada parte de la página.
